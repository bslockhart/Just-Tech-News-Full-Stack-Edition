## Installation
üíæ   The user should clone the repository from GitHub. This application requires Node.js, Express.js, and Sequelize. To connect to the database run mysql -u root -p and enter password from .env file. Then source the schema.sql. To seed the file run npm run seed. Finally to connect to the server run npm start. 
`npm init`
`npm install`
  
## Usage
Run the following command at the root of your project and answer the prompted questions:
`mysql -u root -p`
Enter PW when promted
`source db/schema.sql`
`quit`
`npm run seed`
`npm start`
